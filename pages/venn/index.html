Write an expression in the field to display the corresponding venn diagram. Operators ⋃, ⋂, ∆ and ∖ can be inserted using the buttons. A, B and C can be used to describe general sets, U describes universe. For complements use * before set. Parentheses can be used to decide order of operations. Examples: *B, C∆B, *(A∖U), (A∖*B)⋃*(C⋃A)⋃(B∆*U). For more information on sets see Wikipedia.
